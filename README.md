# boilerplate

## bundle

Boilerplate for React applications:

- create-react-app
- material UI and CSS
- storybook and addons
  - links
  - a11y
  - notes
  - knobs
  - storyshots
  - viewports
  - backgrounds
- testing-library/react

## what would J. do?

He would:

```
git clone
cd boilerplate
npm i
// drink a coffee (or wine ?)
npm start // in one tab
npm t // in another tab
npm storybook // in a 3rd tab
```
